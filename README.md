# uav-sim
an UAV 6-DOF control and simulation model based on MSVC.

## Aircraft Parameters
- **Wing Area:** 1.3536 m²
- **Wingspan:** 3.2 m
- **Mean Aerodynamic Chord:** 0.423 m
- **Thrust-to-Weight Ratio:** 0.25
- **Mass:** 17 kg
- **Moments of Inertia:**
  - Ixx: 1.71 kg·m²
  - Iyy: 5.54 kg·m²
  - Izz: 4.15 kg·m²

## Solving Algorithm:
4-th order Runge-Kutta (RK4)
